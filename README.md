# BreachCore

BreachCore is an in progress Unity game created by me

It is a tower defense afk roguelike project

This repository contains the full Unity source code

## Playing the game

You do not need Unity to play the current build

Go to the Releases page on this repository and download the latest release zip file

Releases page link will be on the right side of this GitHub page or under the Code tab

## About the project

This project is actively developed
Source code changes will be pushed here over time
New builds will be added to the Releases page

If you want to try the game without touching code please download a release build instead of cloning this repository
